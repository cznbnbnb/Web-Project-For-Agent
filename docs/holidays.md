## 添加自定义节假日

在设置页面添加节假日，或直接在项目中配置：

```typescript
project.holidays = [
  "2026-05-01", // 劳动节
  "2026-10-01", // 国庆节
];
```

## 工作日配置

默认工作日为周一至周五：

```typescript
project.workingDays = [1, 2, 3, 4, 5]; // 周一到周日
```

可根据需要调整，例如：

```typescript
// 单双休（大小周）
project.workingDays = [1, 2, 3, 4, 5, 6]; // 包含周六
```
